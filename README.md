## Build Nestjs app with TypeORM

If you without `docker`. You need to install it :tada:


## How to run ?
```bash
$ make up
```

## How to stop and remove container ?

```bash
$ make down
```

## How to stop, remove and destroy saved data in container ?

```bash
$ make destroy
```
